# PPC

Dans la nature on retrouve toutes sortes d’espèces cohabitant ensemble. Il est passionnant de voir comment elle se comporte entre elle. C’est dans cette perspective que nous avons conçu la simulation d’un écosystème composé de prédateur carnivore, de proie herbivore et de l’herbe.
Le but de ce projet est de créer une simulation complète d'un écosystème en Python en utilisant la programmation multi-processus. L'idée est de mettre en pratique ce qu'on a appris en TD sur les systèmes d'exploitation, en faisant tourner plusieurs programmes en même temps qui doivent communiquer entre eux sans se bloquer.


Le code s'exécute en trois étapes. On lance d'abord le gestionnaire (env.py) dans un premier terminal, puis les processus clients (prey.py et predator.py) dans deux autres terminaux séparés.

Au final, ouvrir ces trois terminaux, c’est un peu comme planter les graines d'un monde : on ne lance pas juste du code, on libère un écosystème autonome. C’est là que c'est vraiment gratifiant : la technique pure s'efface pour laisser place au mouvement naturel du vivant, où chaque échange de données entre nos programmes devient le véritable souffle de notre écosystème numérique.
